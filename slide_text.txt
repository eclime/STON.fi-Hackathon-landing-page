--- PAGE 21 ---
Expand your idea into a prompt
Prompt:
Build a web app for a self-custodial RWA
wallet.
Core purpose: A wallet that allows users
to access and manage tokenized U.S. stocks
(xstocks), gold, silver, and stablecoins
without KYC.
Key features:
Users can swap assets, provide liquidity
to earn yield, view token balances, send
and receive assets, and track transaction
history.
Main screen should include:
Portfolio overview with balances, asset list
(xStocks, stables, commodities), quick
actions (Swap, Send, Receive, Earn), and a
simple price chart per asset.
UI requirements:
Dark theme using blue (#0180FF, #3DB1FF) and
purple (#7354F2) accents, clean modern
typography, minimal design similar to a DeFi
wallet dashboard.
note: you can share a link to a webapp design you like
& ask the agent to theme the UI design around it
